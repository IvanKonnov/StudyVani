## 🎯 Что это?

**StudyVani** — полностью автономное веб-приложение для изучения иностранных слов.  
Работает **без интернета**, **без сервера**, **без регистраций** и представляет собой **один HTML-файл**, в который встроены все стили, скрипты и данные.  
Просто откройте файл в браузере — и учите слова где угодно, даже в метро или самолёте.

## ✨ Возможности

### 📥 Импорт слов
- **Загрузка CSV-файла** — подготовьте таблицу в Excel и сохраните как CSV.
- **Вставка из буфера** — скопируйте данные из Excel и вставьте в текстовое поле.
- **Автоматическая генерация примеров** — если не указан пример, приложение само создаст осмысленное английское предложение и его русский перевод.

### 🃏 Умные карточки (Flashcards)
- Все слова идут подряд — никакой лишней навигации.
- Нажмите на карточку, чтобы увидеть перевод и примеры.
- **Два направления обучения:**
  - 🇬🇧 → 🇷🇺 С английского на русский
  - 🇷🇺 → 🇬🇧 С русского на английский
- **Интеллектуальная система повторений:**
  - ✅ «Знаю» — слово уходит в выученные, следующий показ через 7 дней.
  - 🔄 «Повторить позже» — слово вернётся через 5 минут.

### ✅ Тестирование
- Случайные вопросы из всей базы слов.
- 4 варианта ответа (1 правильный + 3 случайных).
- Мгновенная обратная связь с подсветкой правильного/неправильного ответа.
- Примеры использования после каждого ответа.
- Переключение направления теста в одно нажатие.

### 📋 Управление словами
- Удобная таблица со всеми словами.
- **Поиск** по слову или переводу.
- **Фильтры по категориям:**
  - 🆕 Новые (ни разу не отвеченные)
  - 📖 В процессе
  - ✅ Выученные
- **Редактирование на лету** — нажмите на перевод или пример, чтобы изменить.
- **Быстрое удаление** — кнопка с корзиной.

### 🎨 Интерфейс
- 🌸 Приятная фиолетовая гамма.
- 🌓 Светлая и тёмная темы (переключатель в шапке).
- 📱 Полная адаптивность — от монитора до маленького телефона.
- 🎭 Плавные анимации и переворот карточек.

### 📱 PWA (Progressive Web App)
- Установка на главный экран телефона.
- Иконка в меню «Пуск» (Windows) или Launchpad (Mac).
- Полный офлайн после первой загрузки.

## 📖 Инструкция по использованию

### 1️⃣ Подготовка данных для импорта

Слова должны быть в одном из форматов:

**CSV (запятая как разделитель):**

apple, яблоко, I eat an apple every day., Я ем яблоко каждый день.
book, книга
car, машина, a fast car, быстрая машина

**TSV (табуляция как разделитель):**

apple	яблоко	I eat an apple every day.	Я ем яблоко каждый день.
book	книга
car	машина	a fast car	быстрая машина


**Обязательные поля:**
- `english` — слово или фраза на английском.
- `translation` — перевод.

**Необязательные поля:**
- `example_en` — пример использования на английском.
- `example_ru` — перевод примера на русский.

> 💡 **Если примеры не указаны**, приложение сгенерирует их автоматически!

### 2️⃣ Способы добавления слов

**Способ A: Импорт CSV-файла**
1. Подготовьте файл в Excel или Google Sheets.
2. Сохраните как CSV (разделители — запятые).
3. В приложении нажмите **📂 Загрузить CSV**.
4. Выберите файл.
5. Нажмите **👁 Предпросмотр**, чтобы проверить.
6. Нажмите **✅ Импортировать**.

**Способ B: Копирование из Excel**
1. Выделите данные в Excel.
2. Скопируйте (`Ctrl+C` / `Cmd+C`).
3. Вставьте в текстовое поле (`Ctrl+V` / `Cmd+V`).
4. Нажмите **👁 Предпросмотр**.
5. Нажмите **✅ Импортировать**.

**Способ C: Ручной ввод**
1. В текстовом поле напишите строки в формате `english, translation`.
2. Каждое слово с новой строки.
3. Предпросмотр → Импортировать.

### 3️⃣ Изучение слов (карточки)

1. Перейдите на вкладку **🃏 Карточки**.
2. Вы увидите первое слово.
3. **Нажмите на карточку** — она перевернётся.
4. На обратной стороне: перевод и примеры.
5. Оцените своё знание:
   - **✅ Знаю** — слово уходит в выученные (следующий показ через 7 дней).
   - **🔄 Повторить позже** — слово вернётся через 5 минут.

**Переключение направления:**
- Нажмите кнопку `🔄 EN→RU` вверху страницы.
- Она изменится на `🔄 RU→EN`.
- Теперь на карточке сначала русское слово, а перевод — английский.

### 4️⃣ Тестирование

1. Перейдите на вкладку **✅ Тест**.
2. Вы увидите слово и 4 варианта перевода.
3. Выберите правильный вариант.
4. Если ошиблись — правильный ответ подсветится зелёным.
5. Под результатом покажутся примеры использования.
6. Нажмите **Следующий вопрос ➡️**.

### 5️⃣ Управление словами

1. Вкладка **📋 Слова** показывает все слова.
2. **Поиск** — начните вводить слово или перевод, таблица отфильтруется.
3. **Фильтры:** Все / Не выученные / В процессе / Выученные.
4. **Редактирование:**
   - Нажмите на пример (EN или RU) в таблице.
   - Откроется окно редактирования.
   - Измените перевод или примеры.
   - Нажмите **Сохранить**.
5. **Удаление:** нажмите 🗑 → подтвердите.

## 📱 Установка на телефон (PWA)

### Что такое PWA?

PWA (Progressive Web Application) позволяет установить веб-сайт как обычное приложение:
- 🏠 Иконка на главном экране.
- 🪟 Отдельное окно без адресной строки.
- 📴 Полный офлайн-режим.

---

### Android

**Через GitHub Pages (рекомендуется):**
1. Загрузите файл на GitHub Pages (см. раздел ниже).
2. Откройте ссылку в Chrome.
3. Нажмите `⋮` → **«Добавить на главный экран»** или **«Установить приложение»**.

**Локально (продвинутый способ):**
1. Установите приложение **Simple HTTP Server** из Google Play.
2. Положите HTML-файл в папку `Downloads`.
3. В приложении укажите путь к папке, нажмите **Start**.
4. Откройте показанный IP-адрес в Chrome.
5. Нажмите `⋮` → **«Установить приложение»**.

---

### iOS (iPhone / iPad)

**Через GitHub Pages (рекомендуется):**
1. Загрузите файл на GitHub Pages.
2. Откройте ссылку в **Safari**.
3. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой).
4. Прокрутите вниз → **«На экран "Домой"»**.
5. Нажмите **«Добавить»**.

> ⚠️ PWA на iOS работает корректно только через HTTPS (GitHub Pages даёт HTTPS автоматически).

---

## 💻 Установка на компьютер

1. Откройте HTML-файл в Chrome (через GitHub Pages или локальный сервер).
2. В адресной строке появится значок `⊕` (установить).
3. Нажмите на него → **«Установить»**.
4. Приложение появится:
   - **Windows:** в меню «Пуск» и на рабочем столе.
   - **Mac:** в Launchpad.

> ⚠️ **Важно:** PWA-установка не работает при открытии файла через `file://`. Используйте GitHub Pages или локальный сервер.

|---------|----------|

---

### Алгоритм интервального повторения

| Действие | Уровень | Следующий показ |
|---------|---------|-----------------|
| «Знаю» | повышается (0→1→2→3→4→5) | 1 мин → 5 мин → 30 мин → 2 ч → 8 ч |
| «Знаю» (уровень ≥4) | 4 (выучено) | 7 дней |
| «Повторить» | сбрасывается до 0 | 5 минут |
